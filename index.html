The Pythagorean Checker lets you test perfect square numbers and see if they can form a Pythagorean triple.

Rules
  • Enter a whole perfect square number.
  • The application calculates the possible sides A, B, and C of a right triangle using the Pythagorean theorem (a¬≤ + b¬≤ = c¬≤).
  • A, B, and C should not be 0 or negative.
  • If valid, the pythagorean values will be shown along with the average of the sides.
  • Inputs that are not whole perfect square numbers, zero, negative, or cannot form a valid Pythagorean triple will return an "Invalid" result.
